# Documentation Index

Repository documentation is split into two primary areas:

- [Conceptual docs](../docs-conceptual/README.md): research direction, benchmark framing, taxonomy intent, and evaluation philosophy.
- [Technical docs](../docs-technical/README.md): repository implementation, artifacts, scripts, and operational details.

Keep substantive documentation in `docs-conceptual/` or `docs-technical/`; this directory is only an entry point.
